## Installation
1. Click Extensions from acivity bar.
2. Search for `Huacat Pink Theme`.
3. Click **Install** to install it.
4. **Reload** the editor.
5. Please **rate 5 stars!**
6. Click from the menu bar Code > Preferences > Color Theme > Huacat Pink Theme
